# Mapa da Violência e Desenvolvimento Econômico

Projeto acadêmico desenvolvido na Universidade Católica de Santos com o objetivo de analisar a relação entre o Produto Interno Bruto (PIB) e os índices de violência dos municípios brasileiros.

Utilizando dados abertos do IBGE e do Ministério da Justiça, foram realizadas etapas de coleta, tratamento, integração e análise exploratória de dados com Python. O estudo considerou os 100 municípios com maior PIB do Brasil entre 2015 e 2020, gerando estatísticas e visualizações para identificar possíveis correlações entre desenvolvimento econômico e criminalidade.

## Tecnologias Utilizadas

- Python
- Pandas
- Matplotlib
- Plotly
- Google Colab
